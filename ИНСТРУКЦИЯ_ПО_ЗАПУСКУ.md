# Инструкция по запуску для проверяющего

## Минимальные требования

- **Node.js** версии 18 или выше ([скачать](https://nodejs.org/))
- **npm** (устанавливается вместе с Node.js)

## Быстрый запуск (4 команды)

Откройте терминал (PowerShell или cmd) в папке проекта и выполните:

```bash
npm install
npm run db:push
npm run db:seed
npm run dev
```

После запуска откройте в браузере: **http://localhost:3000**

## Что вы увидите

1. **Главная страница** — каталог из 15 товаров с фильтрами и сортировкой
2. **Управление товарами** — кнопка в шапке, CRUD-операции (добавить, изменить, удалить)

## Если что-то не работает

- Убедитесь, что Node.js установлен: `node -v` (должна отобразиться версия)
- Убедитесь, что порт 3000 свободен
- При ошибках БД выполните: `npm run db:reset` и затем снова `npm run dev`
